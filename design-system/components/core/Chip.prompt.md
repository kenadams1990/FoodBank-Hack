Status chip for thermal / QA / urgency readouts — mono text, hairline border, muted tint.

```jsx
<Chip tone="ok" icon="thermometer">3.2°C</Chip>
<Chip tone="alert" icon="thermometer">4.6°C</Chip>
<Chip tone="warn" icon="clock">2d left</Chip>
```

Tones: ok, warn, alert, neutral, accent. State facts ("4.6°C"), never vibes.

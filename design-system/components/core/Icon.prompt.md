Lucide-style line icon at TideLift's documented 1.5px stroke with butt caps — use for any UI glyph.

```jsx
<Icon name="check" size={20} color="var(--foam)" />
```

Documented set: arrow-right, arrow-down, check, close, plus, minus, search, user, settings, bell, menu, info (+ product extras thermometer, clock). For glyphs outside this set, use Lucide CDN at stroke-width 1.5.

# TideLift AI — Demo Script (< 3 minutes)

## Setup
- Dashboard running locally (`npm run dev`)
- Mock surplus feed loaded (3 events: Albacore, Sardine, Crab)

## Minute 0:00 — Surplus Alert
> "A surplus alert just came in from Monterey Bay: 4,200 lbs of Albacore Tuna, 22% below market."
- Show **Forecast Agent** card: species, lbs, discount, estimated can yield (7,560 cans)
- Highlight: **Agent recommends. You decide.** button

## Minute 0:45 — Negotiation Draft
> "The Procure Agent drafted an RFQ. One click to review — you send it."
- Show RFQ: price/lb, expiry, non-profit tax-exempt note
- Judge sees: agent wrote it, human approves

## Minute 1:30 — Canning Slot Booked
> "Richmond co-packer has capacity July 22. Staging plan: 2-day run, 88% yield."
- Show **Canning Agent** staging steps
- Estimated cost per can: $0.38

## Minute 2:00 — Equity Routing
> "Route Agent allocated 7,500 cans: East Oakland gets first priority by protein gap."
- Show delivery plan map / table (neighborhood, cans, window)
- Dietary flags visible: halal, low-sodium, no-shellfish

## Minute 2:40 — NL Brief
> "Any team member can ask: 'What did we move this week?'"
- Show Analyst Agent brief output
- Close: **"Frozen dies. Canned travels. TideLift moves it."**
